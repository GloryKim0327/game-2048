# 2048 经典版

经典的 2048 数字拼图游戏，4×4 网格版本。

---

## 🌐 在线游玩

**https://glorykim0327.github.io/game-2048/**

---

## 🎮 玩法说明

| 操作 | 说明 |
|------|------|
| **目标** | 合并数字，达到 2048 或更高 |
| **移动** | 方向键或滑动屏幕 |
| **规则** | 相同数字碰撞时合并（2+2=4，4+4=8...） |
| **结束** | 无法移动时游戏结束 |

---

## ⌨️ 操作方式

### 电脑
| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 移动方块 |

### 手机
| 手势 | 功能 |
|------|------|
| 上滑/下滑/左滑/右滑 | 移动方块 |

---

## 🎨 方块颜色

| 数字 | 颜色 |
|------|------|
| 2-4 | 浅色 |
| 8-64 | 橙色系 |
| 128-2048 | 金色系 |
| 4096+ | 深色 |

---

## 💡 技巧提示

- 尽量保持最大数字在角落
- 避免频繁上下移动
- 提前规划下一步
